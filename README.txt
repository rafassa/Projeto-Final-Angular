
--COMO RODAR UM SERVIDOR--

para rodar o servidor é nescessario iniciar dois consoles para iniciar o site 

primeiro console:
npm run api 

segundo console:
npm start


--COMO INICIAR UM PROJETO--

para iniciar o projeto é nescessario fazer os seguintes passos:

PASSO-1: 
baixar os node modules com o comando $ npm install

PASSO-2: 
baixar o bootstrap com o comando $ npm i bootstrap@5.3.6

PASSO-3: 
baixar o bootstrap icons com o comando $ npm i bootstrap-icons

PASSO-4: 
configurar o arquivo angular.json colando em scripts:

"node_modules/bootstrap/dist/js/bootstrap.bundle.min.js", 

e ir em angular.json em styles e colando:
"node_modules/bootstrap/dist/css/bootstrap.min.css", 
"node_modules/bootstrap/dist/js/bootstrap.bundle.min.js"

PASSO-5: rodar o servidor, caso não saiba olhe em COMO RODAR UM SERVIDOR

PASSO-6: acessar o servidor http://localhost:4200/ ou a porta que aparecer no seu terminal



--VERSÕES USADAS--

Angular CLI: 19.2.8
Node: v22.14.0
